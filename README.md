## 주요 프로젝트 하이라이트

### LiverGuard CDSS (의료 AI 시스템)

*간암 환자의 병기 예측, 재발 예측, 생존 분석을 수행하는 임상 의사결정 지원 시스템*

**핵심 기술:**
- CT 영상 + 임상 데이터 + mRNA를 활용한 **멀티모달 AI 분석**
- XGBoost, RandomForest 기반 **예측 모델**
- Docker 기반 **마이크로서비스 아키텍처**
